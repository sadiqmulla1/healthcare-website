# Healthcare Portal

A web-based healthcare portal developed using **Java Servlets and JSP**.  
The application allows users to view hospital information, doctors, and gallery images, while administrators can manage doctors and photos through a secure admin panel.

This project demonstrates backend web development using **Java EE technologies** with an MVC-style structure.

---

## Features

### Public Website
- View hospital homepage and services
- Browse doctors and their details
- View hospital photo gallery
- Book appointments

### Admin Panel
- Secure admin login/logout
- Manage doctors (add and delete)
- Upload and delete gallery photos
- View patient appointments

## Running the Project

### 1. Clone the Repository

git clone https://github.com/yourusername/healthcare-portal.git


### 2. Import the Project

Import the project into:

- IntelliJ IDEA
- Eclipse

as a **Maven project**.


### 3. Start MySQL (XAMPP)

- Open XAMPP Control Panel
- Start **Apache** (optional) and **MySQL**

### 4. Configure Database

Update database settings in:
src/main/resources/application.properties

### 5. Deploy to Tomcat

Run the project using **Apache Tomcat**.

### 6. Open in Browser
http://localhost:8080/sample-healthcare
